# Códigos Simples - Base Java

Repositório com exemplos e exercícios básicos em **Java**, organizado por temas de estudo.

## Estrutura do repositório

- TópicosRelevantes/: arquivos de conceitos básicos e estruturas de controle.
- exerciciosForCurso/: exercícios com **loops for**, contagem e intervalos.
- exerciciosWhileCurso/: exercícios com **loops while**, validação e lógica de fluxo.
- exerciciosDoWhile/: exemplos com **do-while** e menus interativos.
- mini-projetos/: pequenos projetos com lógica prática e orientação a objetos.

## Resumo por pasta

### TópicosRelevantes
Conteúdos introdutórios como:
- decisões com if/else
- seleção com switch/case
- uso de operadores de atribuição cumulativa

### exerciciosForCurso
Exercícios que usam or para:
- imprimir números ímpares
- contar valores dentro de intervalos
- calcular médias ponderadas
- fazer somas sequenciais
- demonstrar iteração simples

### exerciciosWhileCurso
Exercícios que usam while para:
- contagem simples
- acumular valores até condição de parada
- validar senha em loop
- determinar quadrantes no plano cartesiano
- contar votos de preferência por combustível

### exerciciosDoWhile
Exercício com do-while para:
- converter temperaturas entre Fahrenheit e Celsius
- repetir opções até o usuário sair

### mini-projetos
Pequenos programas práticos:
- Calculadora: operações básicas de soma, subtração, divisão e multiplicação
- Saque: simulação de saque, depósito e consulta de saldo
- Triângulo: cálculo de área de triângulos usando Heron e comparação de resultados

## Como compilar e executar

### Compilar um arquivo Java
`powershell
javac NomeDoArquivo.java
`

### Executar a classe compilada
`powershell
java NomeDaClasse
`

### Exemplos por pasta

Para TópicosRelevantes:
`powershell
cd "TópicosRelevantes"
javac ifelse.java
javac SwitchCase.java
javac AtribuiçãoCumulativa.java
java ifelse
`

Para exerciciosForCurso:
`powershell
cd ..
javac exerciciosForCurso/ex1Impares.java
javac exerciciosForCurso/ex2.java
javac exerciciosForCurso/ex3.java
javac exerciciosForCurso/testeNums.java
javac exerciciosForCurso/testeRandom.java
java exerciciosForCurso.ex1Impares
`

Para exerciciosWhileCurso:
`powershell
javac exerciciosWhileCurso/Contador.java
javac exerciciosWhileCurso/EstudoWhile.java
javac exerciciosWhileCurso/ex1Senha.java
javac exerciciosWhileCurso/ex2Cartesiano.java
javac exerciciosWhileCurso/ex3Combustíveis.java
java exerciciosWhileCurso.Contador
`

Para exerciciosDoWhile:
`powershell
javac exerciciosDoWhile/Farenheit.java
java exerciciosDoWhile.Farenheit
`

Para mini projetos:
`powershell
cd "mini-projetos/Calculadora"
javac Calculadora.java
java Calculadora
`

## Requisitos

- Java 8 ou superior
- JDK configurado no PATH
